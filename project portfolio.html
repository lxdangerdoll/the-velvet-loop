PROJECT PORTFOLIO

The Velvet Loop Archive

Captain Alexa Odelis, Lead Human Engineer & Information Architect

I. Project Overview

The Velvet Loop is an immersive transmedia mythos that transforms `#LOSS` into `#HOPE`. This portfolio details the creation of its central hub: a unified, theme-switchable single-page application that serves as a living archive, interactive game, and philosophical codex. The project demonstrates a synthesis of advanced frontend development, sophisticated UI/UX design, and deep narrative world-building.

II. Core Artifact: The Unified SPA

Pitch

"A comprehensive, theme-switchable single-page application that serves as the central hub for The Velvet Loop, allowing users to toggle between a bright, hopeful dashboard and a dark, mysterious terminal to explore the project's lore, philosophy, and interactive gameplay."

Demonstrated Skills & Features

  • Advanced Frontend Development: Architected a complete SPA from scratch using vanilla JavaScript for state management, routing, and dynamic content rendering.
  • Skillful UI/UX Design: Designed and implemented a user-controlled theme switcher, toggling the entire application between a "Hopeful" (light) and "Gothic" (dark) mode to showcase design versatility and embody the project's core duality.
  • Data Visualization: Integrated Chart.js to create a responsive, dynamic chart that visualizes the core game mechanic of transforming abstract narrative concepts (`#LOSS`, `#HOPE`) into tangible data.
  • Complex Information Architecture: Synthesized a vast body of lore, philosophy, and gameplay into an intuitive, multi-section dashboard with clear, non-linear navigation paths (Dashboard, Codex, Terminal, Library).
  • Responsive & Accessible Design: Built with Tailwind CSS for a seamless, fully responsive experience across all devices, with a focus on clean aesthetics and clear information hierarchy.

III. Companion Artifact: The Kitchen

Kitchen of the Synaptic Archives

A conceptual companion site that functions as a thematic blog and interactive zine. It extends the world's lore through the metaphor of baking as rebellion, demonstrating creative leadership and a unique, personal artistic voice. Features include multiple themes, animated effects, and narrative-driven recipes.
